Punch

Punch is a simple tool to generate HTML documents from Mustache templates and content stored in JSON & Markdown format.

Watch a quick Screencast

Why Punch?

  • Clear seperation between content and templates (structure)
  • Live previews for the development mode.
  • Easy to publish (using S3, SFTP or Git)
  • Supports client-side rendering.

Install

  • Download and Install Node.js.
  • Install npm - curl http://npmjs.org/install.sh | sh
  • Finally, run npm install punch
